# Contributing to SUBLEQ-CPU

Thank you for your interest in contributing to the **OSH-SUBLEQ-CPU** project!  
This is now a **community-driven** open-source hardware project. We welcome contributions of all kinds.

## Code of Conduct

Please read and follow our [Code of Conduct](./CODE_OF_CONDUCT.md).

## Ways to Contribute

We especially welcome help in the following areas:

- **Verification & Testing** — Writing testbenches, improving coverage, adding formal verification
- **Language Ports** — Porting Verilog to VHDL, SystemVerilog, or other HDLs
- **Examples & Tooling** — Assemblers, program loaders, example programs, hex file generators
- **Architecture Variants** — Harvard architecture, pipelining, interrupts, etc.
- **Peripherals** — Memory-mapped GPIO, UART, timers, etc.
- **Documentation** — Architecture details, tutorials, waveforms, FPGA porting guides
- **FPGA Ports** — Synthesis scripts and constraints for different boards (iCE40, Xilinx, ECP5, etc.)
- **Bug fixes & Refactoring** — Code style, performance, or readability improvements
